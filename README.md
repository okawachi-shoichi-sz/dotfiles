## KeyBind
### tmux

prefix : <C-t>

水平分割 : -  
垂直分割 : \  

ペイン移動  
上 : k  
下 : j  
左 : l  
右 : h

### vim

prefix : s

水平分割 : -  
垂直分割 : \  

新規タブ : t

タブ移動  
上 : k  
下 : j  
左 : l  
右 : h  

タブ幅  
上 : K  
下 : J  
左 : L  
右 : H  

:q : q  
:w : w  
:wq : wq  

ファイルツリー : <Space+n>(prefixなし)  

コードの展開 : <Space+k>(prefixなし)  

Tagバーを表示 : <Space+t>(prefixなし)
